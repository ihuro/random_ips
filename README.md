# random_ips

Random IP addresses generator

## Usage

### Command line

#### Install requirements

    pip install -r requirements.txt

#### Execute command

    python ips.py -h

### Web API

#### Docker

> Install docker and docker-compose https://docs.docker.com/compose/install/

    docker-compose build
    docker-compose up

#### Run local

    pip install -r requirements.txt
    hug -f ips.py
